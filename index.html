Bus Stops In Oakland, CA: Open Steet Map Data vs. AC Transit Data

This map shows areas of Oakland, CA within one-half mile of a bus stop. The impetus for this map was to see what gaps existed between Open Street Map and a potenitally more detailed data set like the one from the AC Transit Agency. QGIS along with the QuickOSM plugin, SpatialLite Database, and the buffer analysis tool were used to create this map.

  • See the map: Bus Stops In Oakland, CA: Open Steet Map Data vs. AC Transit Data
  • Data Sources: AC Transit Data Resource Center and Open Street Map using a key/value pair of highway/bus_stop

Housing Choice Voucher Units By Census Tract In Richmond, California

This map shows the geographic distribution of Housing Choice Voucher Units measured as the percent of all rental units within a census tract in Richmond, California. CARTO was used in the generation of this map

  • See the map: Housing Choice Voucher Units By Census Tract In Richmond, California
  • Data Sources: TIGER/Line Shapefiles for census tracts, American Community Survey for 2014 median household income data, and HUD's eGIS website for housing choice voucher data.
